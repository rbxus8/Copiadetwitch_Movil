📋 Guía Flutter paso a paso
Para implementar en VS Code · Sin experiencia previa
1
Instala Flutter
Ve a flutter.dev, descarga Flutter SDK para tu sistema operativo (Windows/Mac/Linux) y descomprime en una carpeta sin espacios, ej: C:\flutter
2
Instala VS Code + extensión
Abre VS Code → ve a Extensiones (Ctrl+Shift+X) → busca "Flutter" → instala la oficial de Dart Code. También instala la extensión "Dart".
3
Crea el proyecto
Abre la terminal en VS Code (Ctrl+`) y escribe:
flutter create twitch_app
cd twitch_app
code .
4
Reemplaza el código principal
Abre el archivo lib/main.dart y borra TODO su contenido. Pega el código Flutter que te daré a continuación.
5
Ejecuta la app
Con un emulador Android/iOS abierto o tu teléfono conectado, escribe en la terminal:
flutter run
💡 Tip: Si no tienes emulador, instala Android Studio, crea un AVD (virtual device) y luego vuelve a VS Code para ejecutar la app.
